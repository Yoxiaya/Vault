### 踩坑记录

> 图片上传问题

- 问题描述:
  上传图片时，手机端无法获取到图片的File对象，axios无法解析图片数据，导致上传失败。
- 解决办法:
  手动将拼凑File对象，将axios全部替换成fetch，即可上传成功。
